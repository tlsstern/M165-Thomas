# KN-C-03: Programmierung mit Cassandra



**Wenn Sie bereits Erfahrung mit Programmierung mit dieser Datenbank besitzen, führen Sie eine andere Aufgabe durch!**



Greifen Sie auf Ihre Datenbank zu mit einer Programmiersprache. Es geht hierbei nicht darum, dass Sie spezielle Patterns oder Architekturen anwenden, sondern, dass Sie ein Paket/Bibliothek kennenlernen, um auf die Datenbank zuzugreifen.

Sie können z.B. einfach die Abfragen ausführen, die Sie als Skript gespeichert haben aus früheren KNs. 

Verwenden Sie [diese Seite als Einstiegspunkt](https://cassandra.apache.org/doc/latest/cassandra/getting-started/drivers.html) um sich einzulesen und die Programmierung einzusetzen.
