# KN-R-01: Key-Value-Datenbank kennenlernen



Redis ist die momentan meist verbreitete Key-Value-Datenbank und kann auch für in-memory Datenstrukturen verwendet werden, z.B. Caching. 

Sie können Redis als SAAS-Service verwenden oder auch ein Docker-Image installieren. 

Testen Sie die Funktionalität von Redis, z.B. eben die [in-memory Funktion](https://redis.io/docs/latest/develop/get-started/data-store/) oder [Vektor-Funktionalität](https://redis.io/docs/latest/develop/get-started/vector-database/). Testen Sie die CLI, dann auch die Programmierschnittstelle.
